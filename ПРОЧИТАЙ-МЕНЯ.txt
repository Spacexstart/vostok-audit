================================================================
  Сайт vaudit27.ru — короткая инструкция
================================================================

Этот сайт собирается программой Hugo. Чтобы вам не разбираться
с командами, есть программа-помощник.

----------------------------------------------------------------
  КАК ПОЛЬЗОВАТЬСЯ
----------------------------------------------------------------

1. Откройте папку проекта (где лежит этот файл).

2. Дважды щёлкните по файлу:

       ПОМОЩНИК.bat

3. Откроется чёрное окно с меню:

       1) Посмотреть сайт у себя на компьютере
       2) Добавить новый год раскрытия информации
       3) Добавить публикацию в раздел Пресса
       4) Добавить отзыв клиента
       5) Опубликовать изменения (отправить на сайт)
       0) Выйти

4. Введите номер нужного пункта и нажмите Enter.

5. Помощник задаст вам несколько вопросов и всё сделает сам.

----------------------------------------------------------------
  ЧТО НАДО ОДИН РАЗ УСТАНОВИТЬ ДО ИСПОЛЬЗОВАНИЯ
----------------------------------------------------------------

Если помощник пишет красным "не установлено" — нужно один раз
установить программы. Скачайте их по ссылкам и просто пропустите
все шаги установщика по умолчанию:

  • Hugo (программа сборки сайта)
    https://github.com/gohugoio/hugo/releases
    Найдите файл со словом "extended" и "windows" в названии,
    например: hugo_extended_0.147.6_windows-amd64.zip
    Распакуйте hugo.exe в папку C:\hugo
    Добавьте C:\hugo в системный PATH (попросите программиста
    помочь, делается один раз).

  • Git (программа отправки изменений на сайт)
    https://git-scm.com/download/win
    Скачайте, установите со всеми галочками по умолчанию.

После установки помощник заработает сам.

----------------------------------------------------------------
  ТИПИЧНЫЙ СЦЕНАРИЙ: ДОБАВИТЬ НОВЫЙ ГОД РАСКРЫТИЯ
----------------------------------------------------------------

  1) Запускаете ПОМОЩНИК.bat
  2) Выбираете пункт 2 — "Добавить новый год раскрытия"
  3) Вводите год: 2025
  4) Открывается Блокнот с готовым шаблоном
  5) Находите слово "выручка" — заменяете три прочерка ___
     на реальные цифры
  6) Сохраняете файл (меню Файл → Сохранить, или Ctrl+S)
  7) Возвращаетесь в окно помощника
  8) Выбираете пункт 5 — "Опубликовать изменения"
  9) Пишете короткое описание: "добавил год 2025"
 10) Нажимаете Enter
 11) Через 1-2 минуты сайт обновится — открываете
     https://vaudit27.ru/about/disclosure/2025/ и проверяете

----------------------------------------------------------------
  ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК
----------------------------------------------------------------

Если в окне помощника появилось красное сообщение об ошибке —
сделайте скриншот и отправьте программисту. Файлы сайта
автоматически НЕ ломаются — спокойно закрывайте окно и
пробуйте ещё раз.

Подробная инструкция для программиста — в файле README.md
(в этой же папке).
